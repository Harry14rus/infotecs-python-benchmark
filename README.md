  Асинхронная реализация

Программа использует асинхронное программирование для ускорения тестирования:

 Ключевые особенности:
1. Параллельные запросы - все запросы выполняются одновременно
2. Ограничение через семафор - максимум 10 одновременных соединений
3. Асинхронное чтение файлов - не блокирует выполнение
4. Эффективное использование ресурсов - один поток для множества запросов

 Сравнение скорости:
- Последовательные запросы (10 хостов × 5 запросов): ~50 секунд
- Асинхронные запросы (10 хостов × 5 запросов): ~5-10 секунд

 Техническая реализация:
- Используется `asyncio` для управления асинхронными задачами
- `aiohttp` для асинхронных HTTP запросов
- `aiofiles` для асинхронной работы с файлами
- `asyncio.gather()` для параллельного выполнения задач
- `asyncio.Semaphore` для контроля нагрузки
